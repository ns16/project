<h3>Знакомство с ветками</h3>

git branch <имя ветки> – создать ветку с заданным именем. Данная команда только создает новую ветку, но не переключает на нее пользователя.
git checkout <имя ветки> – перейти на ветку с заданным именем.

<h3>Приемы работы с ветками</h3>

Двумя основными приемами работы с ветками являются использование долгоживущих и тематических веток.

Прием тематических веток заключается в создании вспомогательных веток для решения небольших задач. После выполнения такой задачи и слияния с основной веткой вспомогательная ветка удаляется.
